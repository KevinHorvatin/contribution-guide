# Runtime Details

The runtime is a lua virtual machine that is responsible for:
- [Running Lua code, usually after being transpiled from JavaScript](#lua-vm)
- [Maintaining compatibility with Node.js](#node.js)
- [Calling into firmware functions for hardware dependencies](#integration-with-firmware)
- [Providing SSL bindings](#ssl-support)
- [Providing POSIX bindings](#posix-support)
